Struct Reg 

Source 
pub struct Reg(/* private fields */);
Expand description

A register named in an instruction. This register can be a virtual register, a fixed physical register, or a named spillslot (after regalloc). It does not have any constraints applied to it: those can be added later in MachInst::get_operands() when the Regs are converted to Operands.
