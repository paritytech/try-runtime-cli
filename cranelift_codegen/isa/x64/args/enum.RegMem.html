pub enum RegMem {
    Reg {
        reg: Reg,
    },
    Mem {
        addr: SyntheticAmode,
    },
}
Expand description

An operand which is either an integer Register or a value in Memory. This can denote an 8, 16, 32, 64, or 128 bit value.
