fn generate_session_keys( &self, __runtime_api_at_param__: <Block as BlockT>::Hash, owner: Vec<u8>, seed: Option<Vec<u8>>, ) -> Result<OpaqueGeneratedSessionKeys, ApiError>

Generate a set of session keys with optionally using the given seed. The keys should be stored within the keystore exposed via runtime externalities.

The seed needs to be a valid utf8 string.

Returns the concatenated SCALE encoded public keys.
