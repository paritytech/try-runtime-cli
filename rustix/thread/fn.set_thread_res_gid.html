Expand description

setresgid(rgid, egid, sgid)—Sets the real, effective, and saved group ID of the current thread.

§Warning

This is not the setresgid you are looking for… POSIX requires gids to be process granular, but on Linux they are per-thread. Thus, this call only changes the gid for the current thread, not the entire process even though that is in violation of the POSIX standard.

For details on this distinction, see the C library vs. kernel differences in the manual page and the notes in set_thread_gid. This call implements the kernel behavior.
