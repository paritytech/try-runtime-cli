Inform the code buffer of a possible trap at the current location; required for assembling memory accesses.

Source

fn use_target(&mut self, target: DeferredTarget)

Inform the code buffer that a use of target is about to happen at the current offset.

After this method is called the bytes of the target are then expected to be placed using one of the above put* methods.
