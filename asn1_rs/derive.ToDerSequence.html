ยงToDerSequence custom derive

ToDerSequence is a custom derive attribute, to derive both DER Sequence serialization automatically from the structure definition. This attribute will automatically derive implementations for the following traits:

  • ToDer

Serialization will be automatically derived from struct fields. Every field type must implement the ToDer trait.

See derive documentation for more examples and documentation.

ยงExamples

To serialize the following ASN.1 structure:

S ::= SEQUENCE {
    a INTEGER(0..2^32),
    b INTEGER(0..2^16),
    c INTEGER(0..2^16),
}

Define a structure and add the DerSequence derive:

use asn1_rs::*;

#[derive(DerSequence, ToDerSequence)]
struct S {
  a: u32,
  b: u16,
  c: u16
}

let s = S { a: 1, b: 2, c: 3 };
let data = s.to_der_vec().expect("Serialization failed");

ยงDebugging

To help debugging the generated code, the #[debug_derive] attribute has been added.

When this attribute is specified, the generated code will be printed to stderr during compilation.

Example:

use asn1_rs::*;

#[derive(DerSequence, ToDerSequence)]
#[debug_derive]
struct S {
  a: u32,
}
